# MusicStore-IS

Автоматизированная информационная система для магазина музыкальных инструментов.

## Технологии
- C# 10.0
- .NET 6.0
- Консольное приложение

## Функционал
- Учёт товаров, клиентов, продаж
- Оформление продажи с контролем остатков
- Отчёты

## Как запустить
Откройте решение в Visual Studio 2022 и нажмите F5.
